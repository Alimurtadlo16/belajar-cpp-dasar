Learning Basic C++
---
This repository contains a collection of source code and practical exercises from my self-paced learning journey in mastering the fundamentals of programming using the C++ language. This project was built to establish a deep understanding of low-level memory management, syntax structures, and core computational logic.

📌 Project Status: ✅ Completed (Maintained)
All basic core concepts from the introductory materials have been thoroughly implemented and tested. This repository will be continuously maintained and updated as new problem-solving challenges or code optimisations arise.

🚀 Syllabus & Code Features
Within this repository, there are practical implementations covering essential C++ building blocks, including:

Syntax & Fundamental Data Types: Understanding variables, constants, console inputs/outputs (cin/cout), and primitive data types.

Control Flow Structures: Implementing conditional statements (if-else, switch-case) and various looping mechanisms (for, while, do-while).

Functions & Scope: Modularising code by creating custom functions, understanding parameter passing, and managing variable scopes.

Arrays & Basic Memory Concepts: Introduction to storing linear data structures and an early look into memory layout.

🛠️ Tools & Development Environment
Programming Language: C++ (GCC Compiler / MinGW)

IDE/Text Editor: Visual Studio Code / Dev-C++

Version Control: Git & GitHub

💻 How to Run the Code Locally
Prerequisites
Ensure that you have a C++ compiler installed on your computer (such as g++ via MinGW for Windows or Clang for macOS). You can verify if it is ready via the Terminal or Command Prompt (CMD):

```Bash
g++ --version
```

Steps to Compile and Run
Clone this repository to your local machine:

```Bash
git clone https://github.com/Alimurtadlo16/belajar-cpp-dasar.git
```

Open your terminal and navigate to the directory containing the specific .cpp file you wish to run.

Compile the code using the compiler:

```Bash
g++ main.cpp -o main
```

Run the compiled executable programme:
---
Windows: 
```Bash
main.exe
```

---
macOS/Linux:
```Bash
./main
```

---
(Note: Please adjust main.cpp and main to match the actual file name you want to compile).
